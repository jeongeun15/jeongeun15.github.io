팀 프로젝트


NFC를 이용한 물품 보관함 시스템
사용 언어 Java, SQL, HTML, JavaScript, jQuery
개발 도구 Eclipse 2019-09
사용 프레임워크 Spring Framework 4.1.6
작업 기간 2020.12.07 - 2021.01.25 (7주)
프로젝트 설명 Spring Framework를 사용한 팀 프로젝트입니다.
서버단인 Controller와 Model부분을 RESTful Api서버에 구현한 후 AWS을 통해 배포하였습니다.
그리고 사용자/배송원/관리자 서버를 구분하여 Controller, Model, View를 기능에 따라 각 서버에 구현하였습니다.
사용자는 회원가입/로그인 후 물품 보관, 배송, 기간 연장, 다른 사용자에게 결제 키 전달, 결제 등의 기능을 이용할 수 있습니다.
배송원은 회원가입/로그인 후 자신에게 할당된 배송 물품을 확인하고, 수령을 체크하는 기능을 사용할 수 있습니다.
마지막으로 관리자는 로그인 후 모든 회원 조회 및 권한 부여, 모든 보관함 구역 조회 및 등록과 같은 기능을 수행할 수 있습니다.
